# Employee-Perfomance-Review-System
A performance review system that allow to give regards.

## Technology used 
- node.js
- express
- mongoDB
- passport
- ejs and layouts

## Directory Structure

## Get Started 
- clone the project
- open terminal and write this command one by one
- npm install
- npm start
